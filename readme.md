## 🔥 Features

- ✅ **Fully‑typed** (TypeScript)
- 📡 **Stream** image moderation
- 🌐 **URL** image inputs
- ⚙️ **Threshold presets** + custom thresholds
- 🛠️ Helper utilities: `isNSFW()`, `listFlaggedCategories()`

## 📦 Installation

```bash
npm install sightengine-js
# or
yarn add sightengine-js
```
